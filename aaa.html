[보고서1] 엑셀 데이터 분석 및 인사이트 도출 보고서 작성을 위한 데이터 전처리 및 인사이트 도출 프로그램을 개발할거야. 우선은 엑셀 파일을 업로드 할거야. 사용자는 주제를 입력할 수 있는 항목, 엑셀파일에 대한 설명 항목, 요구사항 요청 항목을 입력하면, 해당 엑셀파일을 분석해서 요청항목과 주제에 맞는 인사이트를 도출해줘. 최종 결과물은 워드파일로 다운로드 받을 수 있게 해줘. 다운로드 받은 워드파일에는 마크다운은 제거를 해주고 실제 보고서처럼 핵심 인사이트 보고서로 다운로드 받을 수 있게 해줘. 업로드 된 엑셀파일 전체의 데이터를 읽고 분석해서 만들어줘. [보고서2] 워드파일 업로드후 보고서 생성 워드파일을 업로드 할거야. 워드파일은 데이터를분석 후 인사이트를 도출한 파일이야. 이 워드파일의 내용을 읽고 분석해서 보고서를 작성해줘. 워드 파일 업로드 후 사용자가 목차 생성 버튼을 클릭하면 워드파일의 내용을 분석해서 자동으로 목차를 생성해주고, 사용자가 수정할 수 있게 해줘. 목차가 만들어지고 나면 사용자는 보고서 생성 버튼을 클릭한다. 보고서 생성버튼을 클릭하면 목차에 맞는 내용으로 보고서를 작성해줘. 최종 작성된 보고서는 워드파일로 다운로드 받을 수 있게해주고 최종 결과물은 워드파일로 다운로드 받을 수 있게 해줘. 다운로드 받은 워드파일에는 마크다운은 제거를 해주고 실제 보고서처럼 내용을 풍부하고 전문적으로 만들어줘. [보고서_완성본] 엑셀>워드>보고서 엑셀파일을 업로드 하면 엑셀파일을 분석해서 보고서 주제, 목차를 만들어주는 프로그램을 개발하고싶어. 업로드한 엑셀파일의 전체 데이터를 분석해줘 엑셀파일을 사용자가 업로드 하면 해당 엑셀파일을 분석해서 보고서주제를 생성해줘. 사용자는 보고서 주제를 수정할 수 있어야해. . 주제를 완성한 뒤에 사용자는 '목차생성' 버튼을 클릭해서 엑셀데이터를 분석해서 주제에 맞는 항목들로 목차를 생성해줘. 생성된 목차를 사용자가 수정이 가능하게 해줘. 목차가 생성이 되면 이 목차를 기준으로 보고서를 작성해보자. 사용자가 '보고서작성' 버튼을 클릭하면 이 목차를 기준으로 첨부된 엑셀파일을 기반으로 (첨부된 엑셀파일의 모든 데이터를 읽고 분석해줘.) 보고서를 작성해줘. 내용은 실제 기업에서 보고하는 수준으로 풍부하고 전문적이며, 프로답게 작성해줘 최종완료가 되면 워드 파일로 다운로드 받을 수 있게 해줘.
